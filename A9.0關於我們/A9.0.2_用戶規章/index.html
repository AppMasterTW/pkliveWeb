第３條、管理規則之生效 當使用者於註冊會員按下「註冊」鍵後，即表示使用者使用本公司所提供的相關服務時，同意並願意遵守本「用戶規章」、「服務條款」、「隱私權保護政策」、本公司之會員規範及使用者所在國家之相關法律規定。

第４條、本公司之義務 本公司應對使用者所提供之個人資料予以保密；但依法若有公務機關、檢調單位、司法機關或其他政府機關之強制或命令要求者，本公司就個人資料提供與否將依法處理。

第５條、使用者之義務及責任歸屬事項
  1. 使用者應正確填寫個人基本資料。並請使用者不要使用身分證字號、出生日期、護照號碼、或其他得以直接或間接識別個人的資料做為帳號及密碼。個人資料若有任何變更時，應依照本網站上會員資料修改之相關規定進行。如使用者未依上述規定而發生糾紛時，本公司將不負任何責任。
  2. 使用者對於直播進行中之所有行為及言語，皆應自負法律責任，與本公司無涉。
  3. 使用者不得於本網站上散佈個人政治、宗教信仰及違反公共秩序、善良風俗之言論。
  4. 使用者不得以任何方式修改本網站內容或變更本網站任何程式。
  5. 使用者於本遊戲或網頁中若因其個人之不當行為或使用方式而遭致損害時，其責任歸屬於使用者本人，本公司不負任何責任。

各位親愛的會員，您好！ 歡迎來到PK Live所提供之線上遊戲服務。
為了保障您使用PK Live所提供各項服務及權益，我們將依據進行遊戲服務需要向您蒐集必要的個人資料。請您於提供個人資料之前，詳閱下列個人資料保護告知事項，以維護您權益：


蒐集之目的

為辦理PK Live所提供之遊戲會員服務、線上儲值、遊戲活動及相關行政事宜，確認您使用上述服務而登入之個人資料（姓名、電話、生日、收件地址及電子信箱）。


利用期間

自您使用PK Live所提供之線上服務起至您終止本服務契約止，惟法令另有規定者不在此限。


利用地區

台灣地區


利用對象及方式

僅供您同意之服務目的及PK Live相關人員內部在相關連之範圍內使用及利用，不會任意將您提供的個人資料轉供其他第三人或移作其他目的使用。


行使權利及方式

  • 您就您的個人資料得向PK Live以書面請求行使以下權利：
    1. 請求查詢或閱覽
    2. 製給複製本
    3. 請求補充或更正
    4. 請求停止蒐集、處理及利用或
    5. 請求刪除。但因PK Live執行職務、業務所必需者或法令另有規定者外，PK Live得拒絕。
  • 您可自由選擇是否提供PK Live您的個人資料，但您所提供之個人資料，經檢舉或PK Live發現不足以確認您的身分真實性或無法聯繫到您本人，及其他個人資料冒用、盜用、資料不實等情形，PK Live有權停止提供對您的服務。

隱私權條款

  • 隱私權條款內容，包括PK Live如何處理在用戶使用網站服務時收集到的身份識別資料，也包括PK Live如何處理在商業伙伴與PK Live合作時分享的身份識別資料。
  • 隱私權條款不適用於PK Live以外的公司，也不適用於非PK Live所僱用或管理的人員。
  • 當您要加入PK Live會員時、使用PK Live的產品或服務、瀏覽PK Live網頁、參加宣傳活動或預測遊戲時，我們會依需求請您填寫一些個人資料，也可以從商業夥伴處取得您個人資料。
  • PK Live也自動保留您瀏覽器上的伺服器數值，包括互聯網協定位址(IP Address)、PK LiveCookie中的資料及您要求取用的網頁紀錄。
  • PK Live對於您個人資料之使用，僅在蒐集之特定目的及相關法令規定之範圍內為之。除非經由您的同意或其他法令之特別規定，絕不會將您所登錄的個人資訊揭露、出借或轉售給第三人，或是將您的資料使用於未事先告知你的其他用途。
  • PK Live絕不會未經您同意的狀況下將您的個人資訊任意揭露，但在包括且不限於之狀況除外：
    • 當您在PK Live的行為，違反PK Live的服務條款或可能損害或妨礙PK Live權益或導致任何人遭受損害，而我們相信揭露您的個人資料是為了辨識、聯絡或採取法律行動之必要時。
    • 司法單位或其他有權機關因公眾安全，要求PK Live公開特定個人資料時，PK Live需在司法單位合法的程序，及對PK Live所有會員安全之考量下，做可能必要的配合時。
      （請注意：為了保護會員個人隱私，我們無法為您查詢其他使用者的帳號資料，請您見諒！若您有相關法律上問題需查閱他人資料時，請務必向警調或司法單位提出告訴後，我們將全力配合警調或司法單位調查並提供所有相關資料，以協助警調或司法單位調查！）
  • 當您在PK Live註冊成為會員後，您可以隨時利用您的帳號和密碼更改您原先提供的資料(密碼、收件地址、電子信箱)。但您所能修改之資料，並不包含您以往在PK Live之服務內容記錄。另，當您帳號停權後，您曾參與過的各項服務之活動紀錄並不會隨著帳號的停權而一併消失。
  • 當您使用PK Live各項服務時，可能會需要使用網頁超連結功能。例如：當您使用某項由第三者所提供的服務，或閱讀PK Live所提供的資訊，而該項資訊包括其他網頁的連結功能時，PK Live將無法繼續提供您連結至其他網站後的隱私權保護，但PK Live誠摯提醒您，在您離開PK Live、進入其他網站時，請記得先閱讀該站所提供的隱私權條款，再決定您是否繼續接受該網站的服務。
  • 「Cookies」是一種小小的記錄軟體，當您拜訪設有「Cookies」裝置的網站時，該網站的Server便會自動送出「Cookies」至您的電腦中。「Cookies」的功能是，幫助網站區別到訪者的身份，記錄並儲存到訪者的使用習慣或選擇等，並且擷取到訪者的IP位址。一般會員制的網站，會利用「Cookies」來儲存使用者的ID名稱與密碼、個人網頁、並記錄到訪者所參觀過的網頁等，好為每位會員建立一份上網參觀或購物的歷史清單。但網站並不能利用「Cookies」修改任何到訪者本身的電腦內容。在此提醒您，大部分的瀏覽器會自動接受「Cookies」的儲存，然而您若不願於瀏覽過的網站上留下記錄，現有多數瀏覽器的設計均允許您變更「Cookies」功能。若您關閉瀏覽器的「Cookies」功能，仍能閱讀PK Live網頁的內容；但若您是PK Live會員，使用Cookies將對您在拜訪PK Live時增加許多使用上的便利；換句話說，若您關閉「Cookies」的功能，則往後您每次拜訪PK Live時將無法認出您來，可能便因此失去提供您更多貼心服務的機會。
  • 在PK Live對您的隱私權提供上述保護時，也要在此提醒您，請妥善保管自己的密碼及或任何個人資料，勿將自己的帳號及密碼，提供給任何人。在您完成任何交易或使用個人服務等程序後，務必請記得登出。若您與他人共享電腦或使用公共電腦，也別忘記關閉瀏覽器視窗，以防止他人讀取您的個人資料或信件。避免開啟來路不明的網站及其附件檔案；不要安裝或執行來路不明之程式，或前往外掛網站、論壇等，以防止您資料的外洩與避免電腦遭木馬程式植入之機會。
  • PK Live及直播間管理人員，皆不會主動向任何玩家要求密碼資料，不論是任何理由皆是不被允許的，若發現有此類情況，應立即回報給PK Live進行處理。
  • 請注意：為保障您的權益，PK Live有權進行修改免責聲明以及隱私權條款之法律條款。修正後的條款將刊登於PK Live網站上，並以公告提醒您前往點選閱讀。您已承認並同意此免責聲明以及隱私權條款之法律條款時，即表示您已經接受下列的免責聲明、隱私權條款以及任何後續的修改。
